\doxysubsection{Detailed Description}
Defines functions and constants for enemy tanks "{}\+Intelligent"{} movement. 

In order to make the enemy movement more interesting, the enemy tanks patrol certain waypoints looking for the player. Some of them are hard-\/coded the other waypoints are randomly generated. This file represents our effort to make the movement seem less predictabls and more challenging.
